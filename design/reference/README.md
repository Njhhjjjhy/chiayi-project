# Reference — context, not rules

Everything in this folder is **background only**: research, history, and the
reasoning behind decisions. None of it is a rule, and none of it is the current
truth. When it disagrees with [../decisions.md](../decisions.md),
[../room.md](../room.md), or [../experience.md](../experience.md), those win.

What lives here:

- **Research** — visual and environmental research that informs the design
  (ceiling form vocabulary, sunset colour, materials, site context).
- **History (`archive/` alongside this folder)** — superseded specs and old
  build plans, kept for the record so we remember *why* things changed. Labelled,
  dated, and never to be followed as current instructions.

If you're looking for what's true or decided right now, you're in the wrong
folder — go back up to [../README.md](../README.md).
